QUESTIONS, ANSWERED

The things every fleet manager asks first.

Does it work on any make?
Yes. Because we read the native senders rather than a proprietary data bus, the ECU is make-agnostic — Mercedes, Hino, Isuzu, FAW, Nissan UD, you name it. We load a calibration profile for your exact engine so the thresholds match the manufacturer’s spec, not a generic average.
Will it void my warranty?
No. It’s snap-on, not splice-in. We don’t cut looms, reflash controllers or interfere with anything the OEM warrants. The installation is fully reversible — take it off and the truck is exactly as it was.
What if the truck already has a CAN bus?
Even better. The same unit listens on CAN where it exists and reads native sensors where it doesn’t — so your 2024 rigid and your 1998 workhorse land on the same platform with the same alerts. One fleet, one standard.
How long does fitting take?
Hours, not days. A typical fitment with calibration is done in three to five hours, and the truck is back on shift the same day. Cameras and extra sensors can go on in the same visit.
What does the driver see?
An in-cab dash display with live temperatures, pressures and clear red-line warnings — plus an audible alert before damage is done. The driver sees the same numbers the fleet manager sees, so there’s no argument about what happened.
